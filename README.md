### 变量
- 变量就是可以改变的量
- 变量就是一个盒子，专门储存东西，在程序的执行过程中可以发生该改变的
### 变量的操作
- 给变量赋值 $a='';
- 输出变量echo
- 判断一个变量是否存在 isset();
- 销毁变量 unset();
- 双引号解析变量，单引号不解析变量
- 变量和字符串用点（.）来连接

### 常量
- 常量是程序运行的时候不可改变的量
- 常量的值为标量，作用域为全局，不能写到字符串中
- 系统常量 __FILE__
### 运算符
- 算术运算符：+-*/%
- 赋值运算符：= -= += *= /+ %= .=
- 自增运算符 ++ --
- 比较运算符:< > >= <= == === != !==
- 逻辑运算符：&& 逻辑与并且 有一个为假就是假，只有同时为真才是真；|| 逻辑或，多个同时为假才是假；！就是假变真，真变假
